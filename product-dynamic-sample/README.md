# magic vue: product dynamic sample
* This repo is upgraded product data sample, adding new feature for dynamic display component implemented via Vue-Router.

## Project setup
```
npm install
```

### Compiles and hot-reloads for development
```
npm run serve
```

### Compiles and minifies for production
```
npm run build
```

### Run your tests
```
npm run test
```

### Lints and fixes files
```
npm run lint
```
